# Template for Magic Fi v1 based on Next.js

## Getting started

1. Clone the code:

```
git clone https://github.com/MagicFi-Team/magic_fi.git
cd magic_fi
```

2. Install the dependencies

```
npm install
```

3. Run

```
npm run dev
```

4. Build

```
npm run build
```

5. Create docker image

```
docker build -t magic_fi .
```
